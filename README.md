# GEIA Madrid Dashboard (Streamlit)

App web interactiva para analizar inversión inmobiliaria turística en Madrid.

## Cómo desplegar en Streamlit Cloud

1. Subí este repo a GitHub.
2. Entrá a https://streamlit.io/cloud
3. Create una app nueva eligiendo:
   - Main file: geia_streamlit_app.py
   - Branch: main
4. Deploy.

## Requerimientos
Se instalan automáticamente desde `requirements.txt`.
